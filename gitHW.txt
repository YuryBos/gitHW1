1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Установить Git
Открыть командную строку
Перейти в папку, где я хочу сохранить репозиторий
Выполнить команду "git clone <url>" (<url> берем на github.com, у предварительно созданого нами репозитория)
Подождать завершения процесса.

2 Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Открываю командную строку в нужной мне папке.
Инициализирую Git-репозиторий с помощью git init.
Добавляю файлы с помощью git add ..
Создаю коммит с помощью git commit -m "Initial commit".
Создаю пустой репозиторий на GitHub.com.
Добавляю удаленный репозиторий к локальному с помощью git remote add origin <url>.
Отправляю локальный репозиторий на GitHub с помощью git push -u origin master.

3 Как получить на локальный компьютер изменения с github.com

Открываю командную строку в папке с репозиторием.
Обновляю локальный репозиторий с удаленным с помощью git pull origin master.

4 Какую информацию нам дают команда :
git status - показывает текущее состояние репозитория, включая какие файлы были изменены, но еще не добавлены в следующий коммит, а также какие файлы ожидают коммита.
git log - показывает историю коммитов в репозитории, включая идентификаторы коммитов, даты и времени, авторов и сообщения коммитов.
git remote -v - показывает список удаленных репозиториев, ассоциированных с локальным репозиторием, включая их имена и URL-адреса.
